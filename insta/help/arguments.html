Supported Arguments

userid

Instagram ID of the person who's profile you wish to see.

username

Instagram Username of the person who's profile you wish to see.
The page will connect to Instagram's API and reload the page with the specified username's Instagram ID.

adjust

Fits images to screen, so images are as big as possible, without being cropped by the viewing area's limits.

rows

Specifies how many images should be on each row.
Images are scaled down to 100/value percent.

accessToken (Hash Parameter)

Specifies a different access token to use to view the specified user's images.
This must be passed as a hash parameter. (/#accessToken=ACCESS_TOKEN).
